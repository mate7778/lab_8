# Flutter Reorderable + Dismissible ListView

A simple Flutter app demonstrating a `ReorderableListView` where each item is also `Dismissible`.

## How to run

1. Make sure you have Flutter installed: https://flutter.dev/docs/get-started/install
2. In a terminal, open this folder:

   ```bash
   cd flutter_reorder_dismiss
   flutter pub get
   flutter run
   ```

## Features

- Drag handle to reorder items
- Swipe left to delete items (dismiss)
- Simple and clean UI for learning and experimentation
